# golden-silver-dark-sirens
A python package for processing golden and silver dark sirens

## SDS_bilby.py
Run this code to start the analysis

## prior.py 
Stores prior used in the analysis

## find_gal_in_CI_varying_H0.py
Filter out the galaxies that lie within the 90% credible region, with the option of plotting the galaxies on top of the sky map

## H0_likelihood.py
Computes the H0 likelihood

## H0_posterior.py
Computes the H0 posterior, requires input of precomputed selection function, output as a npz file and also a posterior png
